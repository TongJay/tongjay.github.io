# NVM使用

### 官网

```
https://github.com/nvm-sh/nvm
```

### 列出可用版本

```
nvm list available
```

### 切换node镜像源

```
nvm node_mirror https://npmmirror.com/mirrors/node/
```

### 安装版本

```
nvm install <version>
```

### 切换版本（需要先安装）

```
nvm use <version>
```

```
node_mirror: https://npmmirror.com/mirrors/node/
npm_mirror: https://npmmirror.com/mirrors/npm/
```
